# Wayve Type Scale Tool

A powerful, visual tool for creating responsive fluid typography scales using CSS `clamp()`. Generate harmonious type systems that scale smoothly between viewport sizes with live preview, interactive graphs, and shareable configurations.

## Features

### 🎨 Visual Type Scale Preview
- Live preview of all type scale steps (from XS to 5XL)
- Real-time display of current pixel sizes at your viewport width
- Sample text rendering in Work Sans font

### 📊 Interactive Graph Visualization
- Visual representation of how each type step scales across viewport range
- Color-coded lines for each scale level
- Pixel size labels at minimum and maximum viewports
- Clear markers showing viewport boundaries

### ⚙️ Flexible Configuration
- **Viewport Range**: Set minimum (default: 360px) and maximum (default: 1920px) viewport widths
- **Base Font Sizes**: Define font sizes at min and max viewports
- **Type Scales**: Choose from musical interval ratios:
  - 1.067 (Minor Second)
  - 1.125 (Major Second)
  - 1.2 (Minor Third)
  - 1.25 (Major Third)
  - 1.333 (Perfect Fourth)
  - 1.414 (Augmented Fourth)
  - 1.5 (Perfect Fifth)
  - 1.618 (Golden Ratio)
  - 1.667 (Major Sixth)
  - 1.778 (Minor Seventh)
  - 1.875 (Major Seventh)
  - 2 (Octave)

### 💻 Generated CSS Output
- Automatically generated CSS custom properties
- Syntax-highlighted code display
- One-click copy to clipboard
- Includes URL comment for configuration tracking
- 9 type scale steps: `--step--2` through `--step-6`

### 🔗 Shareable Configurations
- URL parameters automatically update as you adjust settings
- Share exact configurations with colleagues
- Bookmark different type scale presets
- Configuration preserved in generated CSS comments

### 📱 Responsive Design
- Works seamlessly on mobile, tablet, and desktop
- Live viewport width indicator
- Adaptive layouts for different screen sizes

## How It Works

The tool calculates fluid typography using the CSS `clamp()` function. For each type scale step:

1. **Calculate sizes**: Multiply base font size by the scale ratio raised to the step power
2. **Generate clamp()**: Create a fluid formula that interpolates between min and max sizes
3. **Preview**: Display the actual rendered size at current viewport
4. **Visualize**: Show the scaling progression on an interactive graph

### Example Output

```css
/* Generated by: https://your-domain.com/?minWidth=360&maxWidth=1920&minFontSize=18&maxFontSize=20&minScale=1.2&maxScale=1.25 */
:root {
  --step--2: clamp(0.7813rem, 0.7747rem + 0.0326vw, 0.8000rem);
  --step--1: clamp(0.9375rem, 0.9272rem + 0.0516vw, 0.9704rem);
  --step-0: clamp(1.1250rem, 1.0739rem + 0.2273vw, 1.2500rem);
  --step-1: clamp(1.3500rem, 1.2424rem + 0.4780vw, 1.5625rem);
  --step-2: clamp(1.6200rem, 1.4330rem + 0.8315vw, 1.9531rem);
  --step-3: clamp(1.9440rem, 1.6458rem + 1.3262vw, 2.4414rem);
  --step-4: clamp(2.3328rem, 1.8815rem + 2.0058vw, 3.0518rem);
  --step-5: clamp(2.7994rem, 2.1312rem + 2.9708vw, 3.8147rem);
  --step-6: clamp(3.3592rem, 2.3864rem + 4.3252vw, 4.7684rem);
}
```

## Installation & Usage

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/chrishow/wayve-type-scale-tool.git
cd wayve-type-scale-tool
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser to the URL shown (typically `http://localhost:5173`)

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## Using the Generated CSS

Copy the generated CSS and add it to your stylesheet:

```css
/* Your generated type scale */
:root {
  --step--2: clamp(...);
  --step--1: clamp(...);
  --step-0: clamp(...);
  /* ... etc */
}
```

Then use the custom properties in your styles:

```css
.small-text {
  font-size: var(--step--1);
}

.body-text {
  font-size: var(--step-0);
}

.heading-lg {
  font-size: var(--step-3);
}

.display-heading {
  font-size: var(--step-6);
}
```
